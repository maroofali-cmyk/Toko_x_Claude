## Tips for Working with the Agent

1. **Be Specific:** The more context you provide, the better the recommendations
2. **Ask for Research:** Request case studies and examples - the agent will find them
3. **Iterate:** Start with a broad request, then drill down into specifics
4. **Request Options:** Ask for 2-3 strategic alternatives with pros/cons
5. **Challenge Assumptions:** Ask "what if" questions to stress-test strategies
