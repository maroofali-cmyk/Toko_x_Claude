## Appendix B: Content Strategy Notes

### Rupin Brand Keyword Transition

Since Rupin is a new brand, the SEO strategy must account for:

1. **Bridge Keywords:** "Udhaar Book is now Rupin" - capture existing brand searches
2. **Brand + Category:** "Rupin khata app", "Rupin business app", "Rupin ledger app"
3. **New Brand Authority:** Build topical authority around "Rupin" + business management
4. **301 Redirects:** udhaar.pk/udhaarbook/ -> rupin.pk/business-management/ (preserve link equity)
5. **Co-branding Period:** Use "Rupin (formerly Udhaar Book)" in meta descriptions during transition

### Recommended Page URL Structure

```
rupin.pk/                              (homepage)
rupin.pk/business-management/          (main ledger/management page - THIS PAGE)
rupin.pk/credit-book/                  (dedicated credit/khata feature)
rupin.pk/cash-book/                    (dedicated cash book feature)
rupin.pk/salary-book/                  (dedicated payroll feature)
rupin.pk/inventory/                    (dedicated inventory feature)
rupin.pk/invoice-maker/                (dedicated invoice feature)
rupin.pk/earn/                         (dedicated earning features)
```

### Schema Markup Recommendations

- **SoftwareApplication** schema for the main page
- **FAQPage** schema for question-based keywords
- **BreadcrumbList** schema for navigation
- **Organization** schema with EMI license credentials
- **Review/AggregateRating** schema for social proof

### Internal Linking Strategy

Each keyword cluster section should link to its dedicated feature page, creating a hub-and-spoke content model with the business management page as the hub.
